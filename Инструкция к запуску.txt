Проект был собран на сайте spring initializr (https://start.spring.io)
Программа была написана в среде разработки IntelliJ IDEA.

Начать запуск проекта нужно с файла Store.java, находящийся на пути
"src\main\java\com\example\store\Store.java"

Если требуется подключить postgres, нужно:
1. Зайти на File > Settings > Plugins .
2. Скачать плагин DB Navigator.
3. Установить соединение с базой данных postgres (указав пользователя и пароль).
4. Запустить скрипт для создания необходимых таблиц (файл products_store.sql в папке others).
При запуске скрипта может возникнуть проблема, что среда разработки будет искать
файл " psql.exe". Следует самим указать путь к psql.

Если инструмент Maven не загрузил зависимости, нужно в Lifecycle запустить install 
(В окне инструментов IntelliJ IDEA)

Имеется простая коллекция запросов postman в папке others.

Docker.
Перед выполнением команды "docker compose up" в директории src/main/docker выполните следующее:
Выполните команду "mvn clean"
Выполните команду "mvn install"
В директории target скопируйте/перенесите файл store-0.0.2-SNAPSHOT.jar
в в директорию src/main/docker.

Можете выполнить команду "docker compose up"

При этом автоматически создаётся volume "docker_store_db".
